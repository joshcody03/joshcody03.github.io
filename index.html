Introduction

The World Wide Web, invented in 1989 by Tim Berners-Lee, has grown into a crucial part of daily life. The development of web browsers played a key role in shaping the web into what it is today.

Key Terms

  • HTML: Hypertext Markup Language, the standard language for creating web pages.
  • HTTP: Hypertext Transfer Protocol, the protocol for transferring web pages.
  • URL: Uniform Resource Locator, the address used to access web pages.
